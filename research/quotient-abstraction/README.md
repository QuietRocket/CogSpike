# Quotient Abstraction Limitations for Recurrent Competitive Topologies

**Date**: 2026-03-30
**Context**: Investigating why the weight-discretized quotient model fails to verify Winner-Take-All (WTA) dynamics in a 4-neuron contralateral inhibition network, and identifying the minimum quotient configuration that restores correctness.

---

## 1. Problem Statement

The CogSpike weight-discretized quotient abstraction (paper §4) successfully reduces the DTMC state space for feedforward topologies while preserving all PCTL properties. However, when applied to a **contralateral inhibition** network — a recurrent competitive topology where mutual inhibitory connections drive Winner-Take-All dynamics — the standard quotient model produces qualitatively incorrect results.

Specifically, the property `P=? [ F G (y_N2=0) ]` (loser N2 eventually goes permanently silent) evaluates to **1.0** in the precise model but **0.0** in the standard discretized model. This is not a minor quantitative deviation — it is a **qualitative inversion** of the core WTA property.

### Network Topology

The test network has 12 nodes: 4 input neurons (S1–S4), 4 processing neurons (N1–N4) with mutual inhibitory connections, and 4 output neurons (O1–O4). N1 is the predetermined winner due to asymmetric inhibitory weights:

| Edge | Weight | Role |
|---|---|---|
| Si → Ni | +100 | Excitatory input (symmetric) |
| Ni → Oi | +100 | Output projection (symmetric) |
| N1 → N2,N3,N4 | **-100** | Strong suppression from winner |
| N2,N3,N4 → N1 | -70 | Weak back-inhibition to winner |
| Losers ↔ Losers | -70 | Weak mutual inhibition |

The 30-unit asymmetry (100 vs 70) in inhibitory weights is the mechanism that determines the winner.

## 4. Root Cause Analysis

### 4.1 Why the Standard Quotient Fails

The standard quotient model makes two design choices that are individually justifiable for feedforward topologies but jointly catastrophic for recurrent competitive dynamics:

**Problem 1: Non-negative domain (`max(0, ...)`)**

The quotient abstraction maps membrane potentials to equivalence classes in `[0, k]`, clamping negative values to 0. In a contralateral inhibition network, the critical WTA mechanism is **differential inhibition depth**: after mutual suppression, N1 lands at a less negative potential than N2 (because N1 receives -70 per competitor while N2 receives -100 from N1). In the precise model:

```
N1 post-suppression: p = floor(100 - 70 - 70 - 70) = -110
N2 post-suppression: p = floor(100 - 100 - 70 - 70) = -140
```

This 30-unit gap drives differential recovery. With `max(0, ...)`:

```
N1 → max(0, ...) = 0
N2 → max(0, ...) = 0    ← gap erased!
```

Both neurons restart from the same baseline, producing symmetric dynamics with no mechanism for convergence.

**Problem 2: Additive leak (`λ_d = -2`)**

The precise model uses multiplicative leak `r × p` (where `r = 0.5`), which has a crucial dual role:
- **Positive potentials decay** toward 0 (standard leak behavior)
- **Negative potentials recover** toward 0 (the mechanism that creates differential recovery rates)

The additive leak `λ_d = -2` only models the first role. For negative potentials, it pushes them **further negative** instead of recovering them:

```
Multiplicative (precise): p = -110 → 0.5 × (-110) = -55    (recovery)
Additive (quotient):      p = -8   → -8 + (-2) = -10       (worsening!)
```

### 4.2 Why Negative Domain Alone Doesn't Fix It (Disc-W4-neg)

Extending the domain to allow negative values (experiment 2) partially helps: the dominance ratio improves from 1.09× to 1.88× because the initial differential (-7 vs -8 for N1 vs N2) is no longer clamped to 0. However, the additive leak `-2` means:

```
N1 at -7: next tick = -7 + 4 + (-2) = -5    (recovers by +2)
N2 at -8: next tick = -8 + 4 + (-2) = -6    (also recovers by +2)
```

Both recover at the **same rate** (+2/tick). The initial 1-unit gap (-7 vs -8) is preserved but never amplified. In the precise model, the multiplicative leak creates *asymmetric recovery rates*:

```
N1 at -110: next tick = 100 + 0.5×(-110) = 45   (recovery of +155)
N2 at -140: next tick = 100 + 0.5×(-140) = 30   (recovery of +170, but to lower level)
```

The gap *compounds* because recovery is proportional to current depth.

### 4.3 Why Discretization at W=4 with Multiplicative Leak Also Fails

Even with multiplicative leak, W=4 is too coarse. The recovery computation:

```
N1: floor(4 + 0.5 × (-5)) = floor(1.5) = 1
N2: floor(4 + 0.5 × (-6)) = floor(1.0) = 1    ← floor() erases the 0.5 gap!
```

The `floor()` function quantizes away the differential because the weight granularity (W=4) produces post-suppression values (-5 vs -6) whose multiplicative recoveries (1.5 vs 1.0) map to the same integer.

### 4.4 Why W=6 Succeeds

At W=6, δ₆(-100) = -6 and δ₆(-70) = -4 (since round(0.7×6) = round(4.2) = 4). The post-suppression potentials are:

```
N1: 6 - 4 - 4 - 4 = -6
N2: 6 - 6 - 4 - 4 = -8     (gap = 2, vs. gap = 1 at W=4)
```

Recovery with multiplicative leak:

```
N1: floor(6 + 0.5 × (-6)) = floor(3) = 3    → Level 2 → 40% fire
N2: floor(6 + 0.5 × (-8)) = floor(2) = 2    → Level 1 → 20% fire
```

The 2× firing probability differential (40% vs 20%) drives WTA convergence. N1 is twice as likely to fire first after mutual suppression. Once N1 fires and N2 doesn't, N1's -6 inhibition cancels N2's +6 excitatory input, trapping N2 at 0. The system converges to an absorbing BSCC where only N1 fires.

## 5. Necessary and Sufficient Conditions

For a weight-discretized quotient model to preserve WTA dynamics in contralateral inhibition networks, both conditions must hold:

### Condition 1: Multiplicative Leak

The leak must be multiplicative (`r × p`) rather than additive (`λ_d`). This ensures:
- Negative potentials recover toward 0 (rather than diverging)
- Recovery rate is proportional to inhibition depth (creating differential dynamics)

### Condition 2: Sufficient Weight Resolution

The discretization parameter W must be large enough that, after mutual suppression, the multiplicative recovery maps the post-suppression potentials of the winner and loser to **different integer values** that fall in **different threshold levels**.

Formally, for a 4-neuron network with excitatory weight `w_e`, strong inhibitory `w_s`, weak inhibitory `w_w`, and fan-in of 3 inhibitory connections:

```
Post-suppression:
  p_winner = W - (n-1) × round(|w_w| × W / w_max)
  p_loser  = W - round(|w_s| × W / w_max) - (n-2) × round(|w_w| × W / w_max)

Recovery (one tick, no firing, input on):
  rec_winner = floor(W + r × p_winner)
  rec_loser  = floor(W + r × p_loser)

Requirement: rec_winner > rec_loser
             AND they must be in different threshold levels
```

For the specific weights (100, -100, -70) and r=0.5:

| W | p_winner | p_loser | rec_winner | rec_loser | Different levels? |
|---|---|---|---|---|---|
| 3 | -3 | -5 | floor(1.5)=1 | floor(0.5)=0 | ✅ but W=3 too coarse for weights |
| 4 | -5 | -6 | floor(1.5)=1 | floor(1.0)=1 | ❌ |
| 5 | -7 | -8 | floor(1.5)=1 | floor(1.0)=1 | ❌ |
| **6** | **-6** | **-8** | **floor(3)=3** | **floor(2)=2** | **✅** |
| 7 | -8 | -10 | floor(3)=3 | floor(2)=2 | ✅ |
| 10 | -11 | -14 | floor(4.5)=4 | floor(3)=3 | ✅ |

**W=6 is the minimum** for this network configuration.

> **Note**: W=3 appears to produce different recovery values, but the weight mapping δ₃(-70) = round(2.1) = 2 and δ₃(-100) = -3 produces p_winner = 3 - 3×2 = -3 and p_loser = -2×2 = -4... actually the recovery would be floor(3+0.5×(-3))=floor(1.5)=1 and floor(3+0.5×(-4))=floor(1)=1. So W=3 also fails! The table entry is corrected: at W=3, the post-suppression values are -3 and -4, recoveries are 1 and 1 — same level.

## 6. Implications

### 6.1 For the Paper

The current paper presents the quotient abstraction with an additive leak factor `λ_d` and non-negative domain. This works for all feedforward topologies tested (chains, forks, diamonds, convergent). The contralateral inhibition case study uses the **precise model** (not the quotient), so the paper's existing results remain valid.

However, if a future revision discusses applying the quotient to recurrent/competitive topologies, the following must be noted:
- The standard quotient with additive leak is **unsound for WTA properties** in recurrent competitive networks
- A modified quotient with **multiplicative leak** and **negative domain** preserves WTA at the cost of a minimum weight resolution (W ≥ 6 for the tested configuration)
- The modified quotient still achieves substantial state space reduction (50× for this topology)

### 6.2 For CogSpike Implementation

The quotient model generator could be extended with a "recurrent-aware" mode that:
1. Detects recurrent cycles in the SNN graph
2. Switches from additive to multiplicative leak for neurons in cycles
3. Computes the minimum W that preserves recovery differentials
4. Extends the potential domain to include negative values with bounds from fan-in analysis

### 6.3 Theoretical Significance

The finding highlights a fundamental tension in quotient abstractions for stochastic systems: the standard approach of partitioning into non-negative equivalence classes is **incomplete** for systems where negative state values carry competitive information. The multiplicative leak requirement is analogous to requiring that the abstraction preserve the *direction* of the leak dynamics (decay toward zero from both sides), not just its magnitude.
